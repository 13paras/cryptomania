## Getting started

1. To get started with this project, run

```bash
  https://github.com/13paras/cryptomania.git
```

2. Navigate into the project directory:

    ```bash
    cd cryptoMania
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

4. Start the development server:

    ```bash
    npm run dev
    ```
